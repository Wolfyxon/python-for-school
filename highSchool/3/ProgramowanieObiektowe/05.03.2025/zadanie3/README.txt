Zadanie: System Zarządzania Lotniskiem

Napisz program do zarządzania lotniskiem, który będzie obejmować zarządzanie lotami,
samolotami, pasażerami, terminalami itp.
Wymagania:
1. Klasa `Flight`: Klasa reprezentująca lot. Każdy lot powinien mieć numer lotu, trasę, datę i
godzinę odlotu oraz przylotu, typ samolotu itp.
2. Klasa `Aircraft`: Klasa reprezentująca samolot. Powinna zawierać informacje o numerze
rejestracyjnym, typie samolotu, liczbie miejsc itp.
3. Klasa `Passenger`: Klasa reprezentująca pasażera. Każdy pasażer powinien mieć imię,
nazwisko, numer dokumentu, klasę biletu itp.
4. Klasa `Terminal`: Klasa reprezentująca terminal lotniska. Powinna zawierać listę lotów,
samolotów, pasażerów oraz metody do zarządzania nimi.
5. Klasa `Airport`: Klasa reprezentująca lotnisko jako całość. Powinna zawierać listę terminali,
lotów, samolotów, pasażerów oraz metody do zarządzania nimi.
6. Interfejs użytkownika: Napisz interfejs użytkownika umożliwiający zarządzanie lotami,
samolotami, pasażerami, terminalami itp.
Dodatkowe punkty:
- System rezerwacji biletów: Dodaj funkcję rezerwacji biletów dla pasażerów na konkretne
loty, z różnymi klasami biletów, opcjami wyboru miejsc itp.
- Obsługa bagażu: Dodaj funkcję śledzenia bagażu pasażerów, generowania biletów
bagażowych, obsługi reklamacji itp.
- Monitorowanie ruchu lotniczego: Dodaj funkcje monitorowania ruchu lotniczego,
generowania raportów opóźnień, odwołanych lotów itp.
- System logistyki: Dodaj funkcje zarządzania towarami, logistyką załadunku i rozładunku
samolotów, zarządzania paliwem itp.
